# 🎉 Direct File Upload Guide - Welcome Bells Admin Panel

## ✅ What's New

You can now **directly upload images and videos** from your computer! No more copying URLs! 📸

---

## 📸 How to Upload Product Images

### Step 1: Login to Admin Panel
- Go to: https://welcome-bells.preview.emergentagent.com/admin/login
- Username: `admin`
- Password: `admin123`

### Step 2: Add or Edit Product
1. Click **"Products"** in the left sidebar
2. Click **"Add Product"** button (or edit existing product)
3. Fill in the form:
   - Product Name
   - Category
   - Description
   - **Product Image** ← Click "Choose File"
   - Price
   - Stock

### Step 3: Upload Image
1. Click the **"Choose File"** button under "Product Image"
2. Select an image from your computer (JPG, PNG, WebP)
3. You'll see: ✓ Selected: your-image.jpg
4. Click **"Create Product"** or **"Update Product"**
5. The image will upload automatically and appear on your website!

---

## 🖼️ How to Upload Gallery Images/Videos

### Step 1: Navigate to Gallery
1. Login to admin panel
2. Click **"Gallery"** in the left sidebar

### Step 2: Add Media
1. Click **"Add Media"** button (top right)
2. Click **"Choose File"** button
3. Select image or video from your computer
   - **Images**: JPG, PNG, WebP
   - **Videos**: MP4, MOV
4. The system automatically detects if it's an image or video
5. Add a title (optional)
6. Select category (optional)
7. Click **"Add to Gallery"**

### Step 3: Done!
- Your image/video uploads instantly
- It appears on the Gallery page immediately
- Videos show on the home page
- Delete anytime by clicking the delete button

## 💡 Tips for Best Results

✅ **Image Size**:
- Products: 800x800 pixels or larger
- Gallery: 1200x1200 pixels recommended
- Keep file size under 5MB for faster loading

✅ **Image Format**:
- JPG/JPEG - Best for photos
- PNG - Best for logos/graphics with transparency
- WebP - Modern format, smaller file size

✅ **Video Format**:
- MP4 - Most compatible
- MOV - Also supported
- Keep videos under 50MB

✅ **Image Quality**:
- Use high-quality images
- Avoid blurry or pixelated photos
- Good lighting makes a difference

✅ **Aspect Ratio**:
- Square images (1:1) work best for products
- Any aspect ratio works for gallery

## 📱 Where Images Are Stored

- All uploaded files are stored in: `/app/frontend/public/uploads/`
- Each file gets a unique name to prevent conflicts
- Files are accessible via URL: `/uploads/filename.jpg`
- Images work immediately on the live website

## 🔄 Updating Existing Products

1. Go to Products in admin panel
2. Click the **Edit icon** (pencil) next to product
3. The current image is shown
4. To change image:
   - Click "Choose File"
   - Select new image
   - Click "Update Product"
5. New image replaces the old one!

## ❌ Deleting Images

### Delete from Products:
1. Edit the product
2. Upload a new image (replaces old one)
3. Or delete the entire product

### Delete from Gallery:
1. Go to Gallery in admin panel
2. Find the image/video
3. Click the **"Delete"** button
4. Confirm deletion
5. ✓ Image removed from gallery

## ⚠️ Troubleshooting

### Image not uploading?
- Check file size (should be under 5MB)
- Make sure it's a supported format (JPG, PNG, WebP)
- Check your internet connection
- Try a different image

### Image looks distorted?
- Use square images (1:1 aspect ratio) for products
- Resize image before uploading
- Use minimum 800x800 pixels

### Video not playing?
- Use MP4 format
- Keep video under 50MB
- Check if video is corrupted

### Upload button disabled?
- Make sure you've filled all required fields
- Wait for previous upload to complete
- Check if you're still logged in
