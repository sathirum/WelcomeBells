# How to Add Images to Welcome Bells Website

## 📸 Step-by-Step Guide

### Option 1: Add Product Images

1. **Login to Admin Panel**
   - Go to: https://welcome-bells.preview.emergentagent.com/admin/login
   - Username: `admin`
   - Password: `admin123`

2. **Navigate to Products**
   - Click "Products" in the left sidebar

3. **Add or Edit Product**
   - Click "Add Product" button OR click the edit icon on an existing product
   - Fill in all fields including:
     - Product Name
     - Category
     - Description
     - **Product Image URL** ← NEW FIELD!
     - Price
     - Stock
     - Featured (checkbox)

4. **Get Image URL**
   - Upload your product image to:
     - **Google Drive**: Upload → Right-click → Get link → "Anyone with link" → Copy URL
     - **Instagram**: Right-click on image → "Copy image address"
     - **ImgBB.com**: Free image hosting → Upload → Copy direct link
     - **Imgur.com**: Upload → Copy direct link
   
5. **Paste URL and Save**
   - Paste the image URL in the "Product Image URL" field
   - Click "Create Product" or "Update Product"
   - The image will immediately appear on your website!

---

### Option 2: Add Gallery Images/Videos

1. **Login to Admin Panel**
   - Same credentials as above

2. **Navigate to Gallery**
   - Click "Gallery" in the left sidebar

3. **Add Media**
   - Click "Add Media" button (top right)
   - Select Media Type: Image or Video
   - Paste the Image/Video URL
   - Add a Title (optional)
   - Select Category (optional)
   - Click "Add to Gallery"

4. **Your Gallery**
   - Images will appear on the Gallery page
   - Videos will show on the home page video section
   - Delete items anytime by clicking the delete button

---

### Option 3: Add Testimonials

1. **Navigate to Testimonials** in admin panel
2. Click "Add Testimonial"
3. Enter:
   - Customer Name
   - Rating (1-5 stars)
   - Review text
4. Click "Add Testimonial"

## 🔗 Where to Get Free Image URLs?

### Method 1: Google Drive (Recommended)
```
1. Upload image to Google Drive
2. Right-click → Share → Change to "Anyone with link"
3. Copy the link
4. Use this format: 
   https://drive.google.com/uc?export=view&id=FILE_ID
   (Replace FILE_ID with the ID from your link)
```

### Method 2: ImgBB.com (Free Image Hosting)
```
1. Go to https://imgbb.com
2. Click "Upload Image"
3. Select your image
4. Copy the "Direct Link" URL
5. Paste in the admin panel
```

### Method 3: Instagram Images
```
1. Go to your Instagram post
2. Right-click on the image
3. Select "Copy image address"
4. Paste in the admin panel
```

### Method 4: Imgur.com
```
1. Go to https://imgur.com
2. Click "New Post"
3. Upload your image
4. Click "Copy Link"
5. Add .jpg at the end of the URL
```

## 💡 Tips for Best Results

✅ **Image Size**: Use images at least 800x800 pixels for products
✅ **Format**: JPG, PNG, or WebP formats work best
✅ **Quality**: Use high-quality images for better presentation
✅ **Aspect Ratio**: Square images (1:1) work best for products
✅ **File Names**: Use descriptive names before uploading

## ⚠️ Troubleshooting

**Image not showing?**
- Make sure the URL is a direct image link (ends with .jpg, .png, etc.)
- Check if the image is publicly accessible
- Try opening the URL in a new browser tab to verify it works

**Image too large or small?**
- The website automatically sizes images to fit
- For best results, use square images (1:1 aspect ratio)
